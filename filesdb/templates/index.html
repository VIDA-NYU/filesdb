File database for package managers

Available endpoints:

  • /pypi/<project_name>: List versions of the given project
  • /pypi/<project_name>/<version>: List downloads for that project version
  • /pypi/<project_name>/<version>/<download_filename>: List files inside that download, if available
  • /pypi/<project_name>/<version>/<download_filename>/wheel_metadata: Show the wheel metadata from that download, if available
  • /python/import/<name>: List packages providing the given top-level import
  • /files/prefix/<file_prefix>: List files matching a given prefix, and the projects they come from
  • /files/sha1/<sha1_hash> or /files/sha256/<sha256_hash>: List files that have the given digest, and the projects they come from
